## Getting Started
You have a couple of options to start a local server for the data visualization. 
   - **VS Code "Live Server" extension**: Click `Go Live`. It will automatically open your lab in a browser.   
   - **Python 3**: Run `python -m http.server 8000` from your terminal and visit:     
     `http://localhost:8000`
